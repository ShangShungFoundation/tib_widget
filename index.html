This is widget example allowing tibetan spelling of arbitrary tibetan text.

Visit shangshungfoundation.github.io/tib_learn_app/spellchecker to create it

Modifibale parameters:

  • size - 1 coresponds to default page text size
  • color - sets text color, default is black
  • backgroud - sets text backgroud color, default is gray (#DDD)
